Constructor is a special kind of method that is used to initialize objects.

It is used for initializing the instance members when we create the object of a class.

A constructor is  called only one for an object .

Every class must have a constructor , even if it simply relies on the default constructor . 

Constructor can be of two types :
(1) Non-parameterized -> Default Constructor .
(2) Parameterize Constructor .
